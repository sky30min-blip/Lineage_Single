================================================================================
                    리니지 싱글 - 유저 명령어 / 운영자 명령어 안내
================================================================================
※ 명령 접두사는 서버 설정(Lineage.command)에 따라 다르며, 기본값은 "." 입니다.
※ 채팅창에 입력 후 엔터로 실행합니다.

================================================================================
[ 1 ] 유저 명령어 (일반 플레이어)
================================================================================
게임 내 편의를 위해 사용하는 명령어입니다. .명령어 입력 시 이 목록이 안내됩니다.

---- 안내 ----
.명령어     : 서버 명령어 목록(이 문서의 유저 명령어 부분)을 채팅으로 보여준다.
.도움말     : 도움말 HTML 창을 띄운다.

---- 정보 / UI ----
.틱         : 자신의 HP/MP 초당 회복량(틱)을 채팅에 표시한다.
.정보       : 자신의 캐릭터 정보(스탯, 마방 등)를 보여준다. (.캐릭, .케릭, .마방 동일)
.피바       : 캐릭터 위에 HP바(피통) 표시를 켜거나 끈다. 토글.
.버프       : 버프 NPC와 대화한 것처럼 버프를 받는다. (조건/아데나 소모 있을 수 있음)
.버프시간   : 현재 걸린 버프 목록과 남은 시간을 보여준다.
.랭킹       : 전체/클래스별 랭킹, PVP 랭킹 순위를 보여준다.
.마방       : .정보와 동일. 캐릭터 정보 표시.
.나이       : .나이 숫자 — 캐릭터 나이를 15~60 사이로 설정한다.
.서버정보   : 서버 정보를 채팅으로 보여준다.
.시간       : 한국 표준시간을 보여준다.
.리니지     : 게임 내 리니지 시간을 보여준다.

---- 맵 / 시야 ----
.라이트     : 맵 밝기(라이트)를 켠다. 어두운 맵에서 시야 확보.
.맵핵      : .라이트와 동일.

---- 콘텐츠 ----
.보스       : 보스 몬스터 스폰 목록/정보를 보여준다.
.던전       : 기란감옥 등 던전 이용 가능 시간을 보여준다.
.드랍       : .드랍 아이템이름 (축/일반/저주) — 해당 아이템을 드랍하는 몬스터 목록을 검색한다. 띄어쓰기 없이 입력.
.몹드랍     : .몹드랍 몬스터명 — 해당 몬스터가 드랍하는 아이템 목록을 보여준다.

---- 파티 / 혈맹 ----
.파티       : .파티 아이디 — 해당 캐릭터를 파티로 초대한다. (.초대 동일)
.주변파티   : 주변 5칸 이내 캐릭터들을 모두 파티로 초대한다.
.혈맹파티   : 같은 혈맹원을 파티로 초대한다.
.정산       : 파티 베릴(경험치) 분배 설정/정산 관련.
.혈맹가입   : .혈맹가입 아이디 — 해당 캐릭터의 혈맹에 원격으로 가입 신청한다. (원격가입, 원격혈맹 동일)
.직위       : .직위 아이디 등급 — 혈맹원 직위 변경. [0:혈맹원 1:수호기사 2:부군주 3:군주]
.추방       : .추방 아이디 — 혈맹에서 해당 캐릭터를 추방한다.
.전투       : 혈맹 전투(마크) 모드 토글. (.마크 동일)
.지휘       : .지휘 아이디 — 해당 혈맹원을 지휘관으로 지정한다. (부군주 이상만 가능)
.창설       : .창설 혈맹명 — 군주만 혈맹을 창설한다. (아데나 소모 등 조건 있음)
.무인혈맹   : 무인도 혈맹 등록/관련 기능.

---- 수배 / PVP ----
.수배       : 자신을 수배 상태로 등록한다. (레벨 구간별 수배 금액 적용)

---- 파워볼 ----
.배팅       : 파워볼 배팅 관련.
.파워볼홀   : 파워볼 홀 배팅.
.파워볼짝   : 파워볼 짝 배팅.

---- 편의 기능 ----
.오토루팅   : 자동 루팅(줍기) on/off 토글.
.메세지     : 오토루팅 시 채팅 메시지 on/off. (.멘트 동일)
.물약       : 물약 사용 시 멘트 on/off. (.물약멘트, .물약메세지 동일)
.인벤       : 인벤토리 자동 정리. (.인벤정리 동일)
.스왑       : 장비 스왑(세트 교체). (.장비스왑 동일)
.대미지     : 허수아비 대미지 멘트 on/off. (.데미지 동일)
.전투멘트   : 전투 멘트 on/off.
.이펙트     : 이펙트 관련. (.이펙 동일)
.주사위     : 주사위 이펙트를 재생한다.
.사운드     : 사운드 on/off 토글.
.파티멘트   : 파티 메시지 관련. (.파티메세지 동일)
.자동낚시   : 낚시 중일 때 자동 낚시 기능 (서버에서 활성화된 경우).
.자동물약   : 자동 물약 NPC 대화 창을 띄운다.
.칼질       : 자동 공격(칼질) on/off 토글.
.교환       : 캐릭터 간 아이템 교환 관련.
.매크로     : 매크로 관련 기능.
.포인트     : 포인트 조회 등.
.시세       : .시세 아이템이름 — 시장(상점)에서 해당 아이템 시세를 검색한다.
.상점멘트   : 상점 멘트 관련.

---- 거래 게시판 (서버 기능에 따라) ----
.계좌       : 거래 계좌 정보 등록.
.계좌확인   : 계좌 확인.
.거래게시판 : 거래 게시판 보기.
.판매등록   : 판매 등록.
.판매취소   : 판매 취소.
.판매목록   : 판매 목록.
.구매신청   : 구매 신청.
.구매정보   : 구매 정보.
.구매취소   : 구매 취소.
.구매목록   : 구매 목록.
.입금완료   : 입금 완료 처리.
.입금확인   : 거래 완료(입금 확인) 처리.

---- 기타 ----
.후원       : 후원 안내 쪽지를 연다.
.좌표복구   : (유저) 자신의 저장 위치를 본토 등 안전 좌표로 복구한다. (서버에서 허용 시)

================================================================================
[ 2 ] 운영자 명령어 (GM 전용)
================================================================================
GM 권한이 있는 캐릭터만 사용할 수 있습니다. 게임 내에서 ".영자명령어" 입력 시 목록이 안내됩니다.

■ 사용 방법
  - GM 권한: DB characters 테이블의 gm 컬럼이 200 이상인 캐릭터.
  - GM 툴(계정관리 → GM 권한 관리)에서 캐릭터 선택 후 "GM 권한 부여"로 설정.
  - 명령 접두사(기본 ".") + 명령어를 채팅에 입력 후 엔터.

■ 운영자 모드 토글
.영자모드   : GM 모드 on/off. (gm 값 99 ↔ 0 전환) 채팅에 "운영자 모드 활성화/비활성화" 표시.

---- 위치 / 맵 확인 ----
.확인       : 현재 좌표(X,Y), 맵번호, gfx, gfx모드, 헤딩, 타일값, 객체 수 등을 채팅에 출력한다.
.맵        : .확인과 동일.
.위치       : X, Y, 맵번호만 출력. (.좌표 동일)

---- 유저 관리 ----
.고정       : .고정 아이디 — 해당 캐릭터(계정)를 고정 멤버로 등록한다. 고정 멤버 버프 적용.
.선물       : .선물 캐릭터명 아이템명 갯수 인챈트 bless — 해당 캐릭터에게 아이템을 지급한다.
             (bless: 축=0, 일반=1, 저주=2 등)
.전체선물   : .전체선물 아이템명 갯수 인챈트 bless — 접속 중인 모든 캐릭터에게 해당 아이템 지급.
.아이템     : .아이템 한글네임 count enlevel bress 속성종류[1,2,3,4] 1~5단 인형옵션[1,2,3,4,5] 수치
             — 자신에게 아이템 생성. (옵션 등은 서버 코드 참고)
.귀환       : .귀환 번호 — DB 귀환 목록(gm_teleport)의 해당 번호 위치로 순간이동한다.
             번호 없이 입력 시 귀환 목록이 채팅에 출력된다.
.이동       : .이동 X좌표 Y좌표 맵번호 — 해당 좌표/맵으로 순간이동한다.
.영자마크   : GM 마크(표시) 관련.

---- 소환 / 출두 ----
.소환       : .소환 캐릭터명 — 해당 캐릭터를 자신의 위치로 소환한다. (해당 유저가 자신 위치로 이동)
.출두       : .출두 캐릭터명 — 자신이 해당 캐릭터 위치로 순간이동한다.
.전체소환   : 접속 중인 모든 캐릭터를 자신의 위치로 소환한다.

---- 몬스터 ----
.몬스터     : .몬스터 한글네임 count — 해당 몬스터를 자신 위치에 count 마리 소환한다.
.몹정리     : 주변 또는 지정 몬스터 정리(제거).

---- 버프 / 스킬 ----
.올버프     : 자신에게 올버프(전체 버프)를 건다.
.버프       : .버프 캐릭터명 — 해당 캐릭터에게 버프를 건다.
.스킬올마   : .스킬올마 (캐릭터명) — 자신 또는 지정 캐릭터의 스킬을 올마스터로 만든다. 캐릭터명 생략 시 자신.
.라우풀     : .라우풀 수치 (캐릭터명) — 성향(라우풀) 수치 설정. 캐릭터명 생략 시 자신.

---- 제재 (채금 / 차단) ----
.채금       : .채금 캐릭터명 시간(분) — 해당 캐릭터의 채팅을 지정 시간(분) 동안 금지한다.
             로직: 캐릭터에게 ChattingClose 버프를 시간(분)으로 적용하고, DB에도 반영.
.채금해제   : .채금해제 캐릭터명 — 해당 캐릭터의 채금을 해제한다.
.전체채금해제 : 접속 중인 모든 캐릭터의 채금을 해제한다.

.차단       : .차단 캐릭터명 — 해당 캐릭터의 계정을 차단한다.
             로직: characters에서 account 조회 → accounts에서 block_date 갱신, last_ip를 BadIp에 등록 → 해당 계정 접속 중이면 강제 접속 종료.
.차단해제   : .차단해제 캐릭터명 — 해당 캐릭터(계정) 차단을 해제한다. (block_date 초기화 등)
.전체차단해제 : 전체 차단 해제.

---- 회복 / 스탯 ----
.만피       : .만피 (캐릭터명) — 자신 또는 지정 캐릭터의 HP를 풀피로 채운다. 캐릭터명 생략 시 자신.
.만엠       : .만엠 (캐릭터명) — 자신 또는 지정 캐릭터의 MP를 풀엠으로 채운다.
.올피       : 접속 중인 모든 캐릭터의 HP를 풀피로 채운다.
.올엠       : 접속 중인 모든 캐릭터의 MP를 풀엠으로 채운다.
.부활       : .부활 (캐릭터명) — 자신 또는 지정 캐릭터를 부활시킨다. 캐릭터명 생략 시 자신.
.올부활     : 접속 중인 모든 캐릭터를 부활시킨다.
.피뻥       : HP 관련 추가 회복/버프.
.엠뻥       : MP 관련 추가 회복/버프.
.채창       : 채널/채팅창 관련.
.레벨       : .레벨 캐릭터명 레벨.경험치 — 해당 캐릭터의 레벨과 경험치를 설정한다. (예: 52.50 → 52레벨 50% 경험치)

---- 월드 / 기타 ----
.청소       : 바닥에 떨어진 아이템(월드 아이템)을 정리(삭제)한다.
.리로드     : .리로드 항목명 — 서버 데이터 리로드. 항목: 엔피씨, 변신, 스킬, 상점, 보스, 몬스터, 몹드랍, 몹스킬, 백그라운드, 번들, 찬스번들, 서먼, 공지, 서버메세지, 템스킬, 낚시, 팀대전, 타임던전, 로봇, 무인혈맹, 스핵, 몹스폰, 전체스폰, 아이템메세지 등.
.검색       : .검색 [0~2] 이름 — 아이템 검색. 0=잡템, 1=무기, 2=갑옷.
.정보       : (GM) 지정 캐릭터 정보 조회 등.
.상점삭제   : 상점 삭제.
.데미지확인 : 데미지 관련 확인.

---- 날씨 / 이벤트 ----
.날씨      : 날씨 변경.
.투견      : 투견(개경) 이벤트 관련.
.스팟      : 스폰 스팟 관련.
.테베      : 테베 이벤트. (.테베시작 / .테베종료 로 강제 시작·종료 가능)
.악영      : 악영 이벤트.
.지옥      : 지옥 던전 관련.
.얼던      : 얼음 던전 관련.
.콜롯세움   : .콜롯세움 사용/중지 — 기란 콜롯세움 사용 여부 설정.

---- 변신 / 스킬 핵 ----
.변신       : .변신 (변신번호) — 자신의 외형을 해당 gfx로 변경. 기본 1080(메티스).
.유저변신   : .유저변신 캐릭터명 변신번호 — 해당 캐릭터의 외형을 변경한다.
.변신해제   : 변신 해제.
.스핵해제   : 스킬 핵 제외 목록에서 해제.
.스핵제외   : 스킬 핵 검사 제외.
.스핵체크   : 스킬 핵 검사.

---- 검색 / 삭제 ----
.창고검색   : .창고검색 캐릭터명 — 해당 캐릭터의 창고 목록을 보여주기 상점 형태로 연다.
.인벤검색   : .인벤검색 캐릭터명 0(또는 1) — 해당 캐릭터의 인벤토리를 보여준다. (0/1은 탭 구분)
.템검색     : 아이템 검색.

---- 복구 / 기타 ----
.좌표복구   : .좌표복구 캐릭터명 — 해당 캐릭터의 저장 좌표를 본토(안전 좌표)로 DB에서 수정한다.
.몬스터회피 : 몬스터 회피율 관련 설정/확인.
.경험치획등량 : 경험치 획득량 관련.

---- 기타 운영 ----
.봇        : .봇 캐릭터명 — 로봇(봇) 캐릭터의 상태(ID, 모드, 공격 리스트 수 등)를 출력한다.
.포인트지급 : .포인트지급 캐릭터닉네임 포인트 — 해당 캐릭터에 포인트 지급.
.포인트회수 : .포인트회수 캐릭터닉네임 포인트 — 해당 캐릭터에서 포인트 회수.
.메모리정리 : 서버 메모리 GC 실행.
.전체삭제   : .전체삭제 캐릭터명 아이템명 갯수 인챈트 축/저주 — 해당 캐릭터 인벤에서 아이템 삭제.
.인벤삭제   : .인벤삭제 캐릭터명 — 해당 캐릭터 인벤 관련.
.귓말      : GM 귓말 등.
.스텔스    : 투명(스텔스) 모드 토글.
.투망      : 투망(시야) 관련.
.오픈대기   : 서버 오픈 대기 모드.

.셧다운     : 서버 셧다운(종료) 예약 등.

================================================================================
[ 3 ] 명령어 로직 요약 (운영자 명령 예시)
================================================================================

.채금
  사용법: .채금 캐릭터명 시간(분)
  로직: 1) World.findPc(캐릭터명)으로 대상 조회
        2) ChattingClose.init(대상, 시간) — 채팅 금지 버프를 '시간'분 동안 적용
        3) DB에도 채팅 금지 상태 저장
        4) "해당 캐릭터의 채팅을 금지시켰습니다" 메시지 출력

.차단
  사용법: .차단 캐릭터명
  로직: 1) characters 테이블에서 캐릭터명으로 account(계정 id) 조회
        2) accounts 테이블에서 해당 계정의 block_date를 현재 시간으로 갱신 (계정 차단)
        3) last_ip를 BadIpDatabase에 등록 (IP 차단)
        4) 해당 계정으로 접속 중인 캐릭터가 있으면 강제 접속 종료

.귀환
  사용법: .귀환 번호  (번호 생략 시 귀환 목록 출력)
  로직: GmTeleportDatabase에서 번호(uid)에 해당하는 좌표(x,y,map) 조회 후 toTeleport로 이동.
        목록은 gm_teleport 테이블에 등록된 항목들.

.이동
  사용법: .이동 X Y 맵번호
  로직: toTeleport(x, y, map, true) 호출로 해당 좌표로 즉시 이동.

.선물
  사용법: .선물 캐릭터명 아이템명 갯수 인챈트 bless
  로직: World.findPc(캐릭터명)으로 대상 찾은 뒤, toGiveItem으로 해당 아이템을 인벤에 지급.

.소환 / .출두
  .소환 캐릭터명: 대상 캐릭터를 GM 위치로 이동시킴 (toPotal 또는 toTeleport).
  .출두 캐릭터명: GM이 대상 캐릭터 위치로 이동 (toTeleport(pc.getX(), pc.getY(), pc.getMap())).

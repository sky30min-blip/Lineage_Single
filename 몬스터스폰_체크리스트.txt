========================================
몬스터 스폰 관리 기능 - 적용 체크리스트
========================================

[파일 구성]
  ✓ pages/monster-spawn-manager.html - 몬스터 스폰 관리 화면 (gm_tool/pages/)
  ✓ js/monster-data.js               - 몬스터 목록 (서버 monster 테이블 name 과 맞출 것)
  ✓ js/spawn-manager.js              - 쿼리 생성·실행·목록 로직 (pages/ 경로 지원)
  ✓ api/spawn-api.php                - DB 실행/목록/삭제 API (PHP 사용 시)
  ✓ index.html                       - GM 툴 메인 (pages/ 로 링크)
  ✓ config.php.example               - DB 설정 예시 (필요 시 config.php 로 복사)

[사용 방법]
  1. 브라우저에서 D:\Lineage_Single\gm_tool\index.html 열기 (또는 GM툴_실행.bat) → 메뉴에서 "몬스터 스폰 관리" 클릭
  2. 또는 gm_tool\pages\monster-spawn-manager.html 직접 열기
  3. 사냥터/던전 선택 → 스폰 위치 설명 입력
  4. 몬스터 검색 후 선택
  5. 개체수, 리스폰 시간, 좌표(X,Y), X/Y 범위 설정
  6. "쿼리 생성" 클릭 → SQL 미리보기 확인
  7. "DB에 추가하기" 클릭
     - PHP API 사용 시: api/spawn-api.php 가 DB에 직접 실행
     - API 없을 때: SQL이 클립보드에 복사되므로 MySQL 클라이언트에서 붙여넣기

[DB 연동 (선택)]
  - PHP 사용 시: config.php 에 DB 정보 설정 (또는 api/spawn-api.php 상단 기본값 수정)
  - 테이블: monster_spawnlist
  - 서버 스키마가 groups, monster_1~4 등 확장 컬럼이 없으면
    js/spawn-manager.js 의 INSERT 를 기본 11컬럼만 쓰도록 수정 필요

[몬스터 데이터 수정]
  - js/monster-data.js 의 monsterData 배열에서
    name 은 서버의 monster 테이블(MonsterDatabase) 에 등록된 몬스터 이름과 동일해야 함.
  - 서버에 맞게 ID/이름/레벨/타입 추가·수정.

[확인 사항]
  □ 맵 선택 가능
  □ 몬스터 검색/선택 가능
  □ SQL 쿼리 생성 확인
  □ 클립보드 복사 또는 API 로 DB 추가 동작 확인
  □ 맵 선택 시 "현재 맵의 스폰 목록" 표시 (API 연동 시)
